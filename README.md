# 📝 Blogging Platform using Flask

## 📌 Project Overview

This is a full-stack Blogging Platform developed using Flask.
The application allows users to register, log in, create blog posts, and interact with other users through comments, likes, and search functionality.

---

## 🚀 Features

* User Registration and Login (Authentication)
* Create, Read, Update, Delete (CRUD) Blog Posts
* Commenting System with Username
* Like Feature for Posts ❤️
* Search Functionality (by title and content)
* Responsive and User-Friendly Interface
* Live Deployment on Render

---

## 🛠️ Technologies Used

* Python
* Flask
* SQLAlchemy (Database)
* HTML, CSS
* Jinja2 Templating
* Git & GitHub

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://https://github.com/YUVI00111/Blog-Platform.git
```

2. Navigate to the project folder:

```
cd Blog-Platform
```

3. Create virtual environment:

```
python -m venv venv
```

4. Activate virtual environment:

```
venv\Scripts\activate
```

5. Install dependencies:

```
pip install -r requirements.txt
```

6. Run the application:

```
python app.py
```

7. Open in browser:

```
http://127.0.0.1:5000
```
